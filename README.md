# Kimi Skill 科研写作流水线

> 本仓库包含 5 个教学/申报双模式 Kimi Skill，帮助学生和研究者完成从选题到完整课题申报活页的全流程。

---

## 快速开始

### 如果你是第一次使用

1. 确认 Kimi 已加载本仓库中的 skills（每个 skill 文件夹中的 `SKILL.md`）
2. 从 `topic-to-idea` 开始，按顺序执行流水线
3. 每个 skill 默认输出**教学版**，如需申报版，请明确说明"我要申报国家社科基金"或"请输出申报版"

### 如果你已有研究材料

| 你已有的材料 | 从这里开始 | 使用的 skill |
|:------------|:----------|:------------|
| 只有一个模糊的研究兴趣 | topic-to-idea | 生成结构化选题说明 |
| 已有选题说明和理论视角 | literature-to-design | 直接生成研究设计 |
| 已有选题说明、读书报告、文献综述、研究设计 | proposal-writing | 整合为完整活页 |

## Skill 速查表

| Skill | 触发条件 | 教学版核心功能 | 申报版核心功能 |
|:------|:---------|:--------------|:--------------|
| **topic-to-idea** | "帮我生成一个选题说明" "给我一个研究设想" | 随机选题，固定5句格式，≤300字 | 用户指定主题，文献缺口，理论空白 |
| **idea-to-perspective** | "帮我设计读书报告" "理论视角有哪些" | 3选1对抗性阅读，500-800字 | 匹配申请人理论积累，避免盲目3选1 |
| **perspective-to-literature** | "帮我写文献综述" "怎么检索文献" | 双语检索，6格地图，3假设，2000-3000字 | 压缩至1000-1200字，前期成果整合 |
| **literature-to-design** | "帮我设计研究方案" "怎么做实证研究" | 纯网络采集，基础统计，6-8周，≤100元 | 3年周期，自然实验，混合方法，因果识别 |
| **proposal-writing** | "帮我写活页" "整合所有材料" | 拼接+改写+增补，逻辑一致性 | 评审导向，字数精修，匿名合规，前期成果 |

## 双模式说明

每个 skill 支持两种输出模式：

- **教学版（默认）**：面向本科生或初学者，强调可执行性、技术门槛低、时间短
- **申报版**：面向国家社科基金/课题申报，强调方法严谨、评审竞争力、匿名合规

切换方式：在调用 skill 时明确说明"我要申报""请输出申报版""这是课题申报"等。

## 使用示例

### 教学版示例

```
用户：给我一个社会治理方向的选题说明。
Kimi：[调用 topic-to-idea 教学版]
     → 生成 300 字以内选题说明

用户：基于这个选题，设计读书报告。
Kimi：[调用 idea-to-perspective 教学版]
     → 提供 3 个对抗性理论视角，用户选择后生成 500-800 字读书报告要求

用户：基于这个选题和视角，写文献综述。
Kimi：[调用 perspective-to-literature 教学版]
     → 双语检索 + 6 格地图 + 3 个竞争性假设 + 2000-3000 字综述

用户：基于文献综述，设计研究方案。
Kimi：[调用 literature-to-design 教学版]
     → 纯网络采集 + 基础统计 + 6-8 周方案

用户：整合所有材料，写开题报告。
Kimi：[调用 proposal-writing 教学版]
     → 完整 6 章开题报告
```

### 申报版示例

```
用户：我要申报国家社科基金，研究网格化管理与社会治理，
     前期有 3 篇 CSSCI 论文，分别是……
Kimi：[调用 topic-to-idea 申报版]
     → 生成文献缺口导向的选题说明

用户：我倾向于用 Scott 的清晰化理论。
Kimi：[调用 idea-to-perspective 申报版]
     → 匹配申请人理论积累，深化理论框架

用户：请帮我写选题依据。
Kimi：[调用 perspective-to-literature 申报版]
     → 压缩至 1000-1200 字，整合前期成果

用户：请帮我设计研究方案。
Kimi：[调用 literature-to-design 申报版]
     → 3 年周期 + 自然实验 + 混合方法 + 因果识别

用户：请帮我整合成完整活页。
Kimi：[调用 proposal-writing 申报版]
     → 评审导向 + 字数精修 + 匿名合规 + 3 分钟评审测试
```

## 注意事项

1. **教学版**默认生成，适合课程作业、本科生科研训练、开题报告
2. **申报版**需明确说明，适合国家社科基金、省部级课题申报
3. 每个 skill 的**输出格式**严格匹配 NSSFC 活页标准，可直接复制到 Word
4. 活页总字数**不超过 7000 字**，其中选题说明**不超过 300 字**
5. **匿名合规**：活页中不得出现申请人姓名、学校、院系等信息
6. 流水线的 skills 可以**单独使用**，也可以**串联使用**

## 背景与迭代

本仓库 skills 的设计基于：
- 国家社科基金 2025 年度课题指南
- NSSFC 通讯评审意见表评分标准（选题 30%、论证 50%、研究基础 20%）
- 6 份成功立项的申请书/活页样本分析
- 本科生社会科学研究方法课程教学需求

持续迭代方向：
- 增加更多成功/失败案例作为正反示例
- 优化申报版的前期成果匹配算法
- 增加学科特异性模板（社会学、政治学、公共管理等）
