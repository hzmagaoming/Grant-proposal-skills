# Scholar-Book Pool: Curated Canonical Works by Tradition

> `idea-to-perspective` skill 的参考文件。在将学者与学生选题匹配时加载。
> 目的：避免重复推荐安全区作品（平台治理、监管失灵、技术官僚）。

## How to Use This Pool

1. 识别学生的现象类型（劳动、空间、信息、国家、市场、文化等）
2. 从下方不同行中选取三个传统
3. 每个传统挑选一位学者-书籍配对
4. 确保至少有一对挑战学生的默认假设

## Selection Rules

1. **避免热点聚集**：不要全部从 Platform Capitalism、Surveillance Capitalism 等流行话题中挑选。
2. **要求时间深度**：至少有一部书应出版超过 30 年，以确保传统深度。
3. **强制对抗性**：至少一位学者的立场应与学生默认的框架相矛盾。
4. **匹配可及性**：对于入门课程，优先选择有中文译本且页数 <300 的书籍。
5. **禁止教材**：不接受入门综述、百科全书或论文集。
