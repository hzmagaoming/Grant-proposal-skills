## 已确认的研究要素

- **研究问题**：网格化治理是否通过标准化行政程序系统性地替代了基于"米提斯"的民间调解智慧？
- **选定假设**：H2 — 网格化治理用标准化行政程序替代了基于"米提斯"的民间调解智慧，导致复杂矛盾的调解质量下降、复发率上升。
- **理论视角**：James C. Scott, 《国家的视角》, 清晰化理论
- **核心机制**：清晰化逻辑将复杂社会关系压缩为可计算指标，从而排斥不可编码的地方性知识
- **关键概念**：清晰化（legibility）、米提斯（mētis）、表演性合规

---

## 3.1 [研究内容]

### 3.1.1 课题研究的主要目标

本研究试图回答以下三个递进问题：
1. 网格化治理的政策文本在多大程度上体现了"清晰化"逻辑——即通过标准化流程、量化指标和时限要求将社区矛盾转化为可计算事件？
2. 在公开报道的网格调解案例中，标准化行政程序与基于"米提斯"的非正式协商各自占据多大比例，二者的替代关系在文本中如何呈现？
3. 网格化标准化程度较高的地区与较低的地区，居民在社交媒体上的治理信任反馈是否存在系统性差异？

### 3.1.2 重点难点

| 重点 | 难点 |
|:-----|:-----|
| 构建可操作的"清晰化程度"测量指标，使其能够捕捉政策文本中的标准化逻辑 | 政策文本表述存在"形式化"倾向，实际执行可能与文本规定存在偏差 |
| 从新闻报道和社交媒体中提取有效的调解过程信息，区分"标准化程序"与"非正式协商" | 公开文本往往侧重结果而非过程，调解细节常被省略 |
| 建立"网格化标准化程度"与"居民信任反馈"之间的经验关联 | 社交媒体数据存在选择偏差，发帖居民未必代表全体居民 |

### 3.1.4 研究计划及其可行性

**数据采集方案（纯网络、低成本）**

| 项目 | 说明 |
|:-----|:-----|
| 数据来源 | ① 各市政府网站公开的网格化管理实施方案、考核办法（免费）<br>② 新闻报道中关于网格调解的具体案例（百度新闻搜索，免费）<br>③ 社交媒体公开讨论：微博"网格员""社区调解"话题、知乎相关问答（免费）<br>④ 各地政法委/司法局公开的年度矛盾纠纷化解统计报告（免费） |
| 采集字段 | 政策文本：考核指标数量、流程步骤数、办结时限要求、回访机制<br>新闻案例：矛盾类型、调解方式（标准化/非正式）、参与主体、调解结果<br>社交媒体：发帖者身份、情感倾向、对网格调解的具体评价 |
| 采集方法 | 手动下载政策文本 + 手动检索新闻案例 + Python requests 抓取社交媒体公开页面（如需） |
| 样本策略 | purposive 抽样：选取东部、中部、西部各 2 个城市，共 6 个城市；每个城市选取 1-2 个有公开报道的社区案例 |
| 预期样本量 | 政策文本 6-12 份；新闻案例 30-50 个；社交媒体帖子 200-500 条 |
| 时间窗口 | 2020-2024 年（网格化全面铺开后五年） |
| 伦理注意 | 仅采集公开数据，不涉及个人隐私；社交媒体数据仅分析公开可见内容，不追踪个人身份 |

**自然实验锚点**

本研究利用网格化治理推进的**时间断点差异**作为准实验设计：处理组为2018年前已全面铺开网格化治理的城市（如北京、上海、浙江部分城市），对照组为2020年后才全面铺开的城市（如部分中西部城市）。在同一时间截面（2024 年）比较两组的居民反馈差异。

**分析方案（三层结构）**

第一层：描述性发现（所有学生必须完成）
    - 样本城市网格化管理规范的基本特征（频数、均值、标准差）
    - 考核指标数量的分布（直方图、箱线图）
    - 调解模式的类型分布（饼图、条形图）

第二层：推断性分析（有统计基础的学生完成）
    - 组间差异检验：高清晰化城市 vs 低清晰化城市的居民反馈情感得分差异（t 检验 / Mann-Whitney U）
    - 相关分析：考核指标数量 vs 居民满意度（Pearson / Spearman）
    - 内容分析的信度检验：两名编码员独立编码，计算 Cohen's Kappa

第三层：多元分析（统计能力较强的学生选做）
    - 简单线性回归：居民满意度 = β₀ + β₁(清晰化程度) + β₂(城市 GDP) + β₃(人口密度) + ε
    - 子样本比较：按地区（东部/中部/西部）分组回归

**时间线（8 周）**

| 周次 | 任务 |
|:-----|:-----|
| 第 1-2 周 | 确定 6 个样本城市，下载政策文本，建立编码手册 |
| 第 3-4 周 | 检索并整理新闻案例，进行调解模式编码 |
| 第 5 周 | 采集社交媒体数据，进行情感分析/主题编码 |
| 第 6 周 | 数据清洗与描述性统计 |
| 第 7 周 | 推断性分析与回归分析 |
| 第 8 周 | 撰写研究报告，完成质量检查清单 |

**可行性论证**

本方案完全满足"本科生可执行"的四可标准：可理解（政策文本分析和内容分析是本科生基础训练），可操作（主要工具为 Excel 和 Python 基础库，已有大量在线教程），可负担（零成本，所有数据均为网络公开信息），可验证（政策文本和政府报告可通过官方渠道交叉验证）。

## 3.2 [创新之处]

### 3.2.1 学术观点创新

本研究首次将斯科特"清晰化"理论系统引入中国网格化治理研究。现有文献要么 celebrate 网格化的国家能力建设功能（英文主流），要么批评其行政化倾向（中文主流），但都未将网格化理解为一种"认识论暴力"——即行政分类对社会真实的系统性替代。本研究用斯科特的透镜重新框定了网格化治理的研究议程。本研究提出"调解质量的空间压缩"概念，指标准化流程将三维的社会关系（时间维度、情感维度、关系维度）压缩为一维的行政事件（发现→上报→流转→办结），为理解基层治理失灵提供了新的分析工具。本研究直接挑战 Tang (2020) 和 Hu & Zhang (2023) 的"技术提升治理效能"叙事，提出竞争性解释——调解质量的下降不是技术失败的副产品，而是清晰化逻辑成功运作的必然结果。

### 3.2.2 研究方法创新

本研究采用纯网络公开数据，零实地调研、零访谈、零差旅，这在网格化治理研究中较为罕见——现有研究多依赖田野调查，成本高昂且难以复制。本研究展示了如何利用政策文本、新闻报道和社交媒体数据来检验质性理论假设。本研究利用网格化推进的时间断点差异作为准实验设计，为观察性研究提供了因果识别的初步锚点。本研究使用 AI 辅助编写文本分析脚本和进行初步编码，降低技术门槛，使本科生能够处理中等规模的文本数据。本研究的方法（政策文本编码→新闻案例内容分析→社交媒体情感分析）可应用于其他研究问题，具有较强的工具迁移价值。
