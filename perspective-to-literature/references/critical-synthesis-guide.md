# 批判性文献综述写作指南

> 本指南供 `perspective-to-literature` skill 在 PHASE 4 调用。核心目标：撰写一篇 2000–3000 字的批判性文献综述，不是罗列文献，而是展示文献之间的对话、张力与盲区。

---

## 核心原则

### 1. 综述是论证，不是目录

**错误写法**："A(2010)发现X，B(2012)发现Y，C(2015)发现Z……"

**正确写法**："A(2010)声称X，但B(2012)的样本显示X只在特定条件下成立；更重要的是，C(2015)的方法论批评指出，A和B的测量工具都无法捕捉Z，而这恰恰是理解该现象的关键。"

### 2. 每段必须有批判性论点

综述中的每个段落都应该回答一个问题：
- **这段的批判性论点是什么？**
- **如果删掉这段，我会丢失什么论证？**

如果一段只是在描述文献内容而没有评价、对比或指出盲区，就应该删除或重写。

### 3. 中英文文献是对话者，不是分栏陈列

**错误结构**："英文文献发现……中文文献发现……"

**正确结构**："英文文献普遍假设X（A, B, C），但中文研究D在情境E中发现X并不成立，反而支持Y；然而D的方法论选择（F）使其结论难以推广到G情境，这正是我的研究要解决的问题。"

### 4. 用学生已选的理论视角重新阅读文献

综述不是中立的。学生已经通过 `idea-to-perspective` 选择了一个理论视角（如 ANT、空间社会学、道德经济等）。综述应该用该视角作为"透镜"去审视文献：
- 哪些文献的核心概念与该视角兼容？
- 哪些文献的盲区恰好是该视角能照亮的？
- 哪些文献过度引用了某个概念却从未真正检验它？

---

## 综述结构（对话式六段结构）

### §1 引言（约 150–200 字）

**功能**：提出经验 puzzle，说明为什么需要综述这个领域。

**必须包含**：
1. 一个具体的经验现象（不是抽象主题）
2. 这个现象为什么令人困惑
3. 现有文献如何（未能）解释它

**示例**：
> 幽灵外卖——即无实体门店、仅靠平台接单的外卖商家——在中国一、二线城市迅速蔓延。市场监管部门多次开展专项整治，平台也上线了证照审核系统，但问题始终反复。这种"治理—反弹—再治理"的循环表明，现有研究对平台、商家与监管者之间的互动机制缺乏系统理解。

---

### §2 英文文献中的核心发现与盲区（约 400–500 字）

**功能**：梳理英文文献的共识与分歧，指出其理论和方法论盲区。

**写作策略**：
- 按"主张—挑战—盲区"组织，不是按作者或时间
- 每句话都应该是"文献A声称……，但文献B质疑……，因为……"
- 明确指出：英文文献的盲区是什么？它们假设了什么而没有检验？

**段落模板**：
```
英文文献在[主题]上形成了[几派]主要观点。[学派一]（A, B）主张[核心观点]，
其证据主要来自[数据来源/方法]。然而，[学派二]（C, D）质疑这一主张，
指出[限制条件/反例]。更具批判性的是，E(年份)的方法论审查发现，
上述所有研究都共享一个未检验的假设：[假设内容]，而这一假设在[情境]
中可能并不成立。
```

**批判性检查**：
- [ ] 这段是否展示了至少两个文献之间的张力？
- [ ] 这段是否指出了一个未被检验的假设或盲区？
- [ ] 这段是否说明了为什么这个盲区对我的研究重要？

---

### §3 中文文献中的独特贡献与局限（约 400–500 字）

**功能**：展示中文文献在什么问题上有独特贡献，在什么问题上有局限。

**写作策略**：
- 突出"中国情境"带来的特殊发现
- 明确中文文献与英文文献的关系：确认、修正、还是矛盾？
- 指出中文文献的方法论局限（样本偏差、数据来源单一等）

**段落模板**：
```
中文文献对[主题]的研究呈现出与英文文献不同的关注焦点。F(年份)和G(年份)
基于[中国数据来源]发现[独特发现]，这一发现与英文文献中的[某主张]形成
[对比/补充]。然而，中文研究在方法上存在明显局限：现有研究几乎 entirely
依赖[数据来源]，导致[偏差]。更重要的是，H(年份)虽然指出了[重要问题]，
但其分析框架仍然 borrow from [西方理论]，未能充分解释[中国特有现象]。
```

**批判性检查**：
- [ ] 这段是否识别了至少一个中文文献的独特贡献？
- [ ] 这段是否指出了中文文献的方法论局限？
- [ ] 这段是否说明了中文文献与英文文献的理论关系？

---

### §4 理论对话：已选视角如何重新组织文献（约 500–600 字）

**功能**：用学生通过 `idea-to-perspective` 选择的理论视角，重新组织前述文献，揭示新的理解方式。

**这是综述的核心段落，必须最用力写。**

**写作策略**：
1. 明确指出：从[理论视角]来看，现有文献的组织方式有什么问题？
2. 用该视角的核心概念重新解读文献中的发现
3. 指出：哪些文献被过度引用但其核心概念从未被检验？
4. 指出：哪些文献的盲区恰好是该视角能照亮的？

**段落模板（以 ANT 为例）**：
```
从行动者网络理论（ANT）的视角来看，上述中英文文献共享一个根本性的盲区：
它们将监管失败视为"空间消失"或"法律漏洞"的结果，而将平台、商家和监管者
视为具有固定利益和能力的原子化行动者。ANT 则提醒我们，监管失败可能不是
"够不到"，而是"组装不上"——即各方未能将"食品安全"翻译为一个共享目标
（Callon, 1986; Latour, 2005）。

循此视角重读文献，我们发现：英文文献中关于"平台治理"的讨论（A, B, C）
虽然频繁引用"利益相关者"概念，却从未真正追踪这些利益是如何被转译、
固化和挑战的。中文文献中关于"监管困境"的诊断（D, E）同样预设了监管者
和平台之间的对抗关系，却忽略了二者可能在某些节点上形成的临时联盟。

更为关键的是，现有文献将"证照"视为一种静态的法律凭证，而 ANT 视角下，
证照是一种"铭刻"（inscription），其效力取决于它在网络中被承认和流通的
方式。这意味着，即使平台上线了电子证照系统，如果这一铭刻未能与监管者的
执法程序、消费者的信任机制形成连贯的网络，它就可能沦为一种"表演性合规"
（performative compliance）。这一洞见在现有文献中完全缺席。
```

**批判性检查**：
- [ ] 这段是否使用了学生已选理论的核心概念（至少 2 个）？
- [ ] 这段是否重新组织了§2和§3中的文献，而非简单重复？
- [ ] 这段是否识别了至少一个"文献中被过度引用但从未检验"的概念？
- [ ] 这段是否识别了至少一个"只有该视角才能看到的盲区"？

---

### §5 批判性综合：文献为何不足以回答我的问题（约 300–400 字）

**功能**：收束前述分析，明确指出文献的共同盲区，论证为什么需要新的研究。

**写作策略**：
- 不是"这个主题研究很少"（太弱）
- 而是"文献在[特定方面]的共识建立在[未检验假设]之上，当[条件变化]时，这一共识可能崩溃"
- 明确指出中英文文献的共同盲区

**段落模板**：
```
综合上述分析，现有文献——无论中文还是英文——在解释[具体现象]时存在
三个相互关联的盲区。第一，[盲区一]，这导致[后果]。第二，[盲区二]，
中英文文献在此存在[分歧/共识]，但双方都忽略了[关键问题]。第三，也是
最根本的，[盲区三]——现有研究将[某概念]视为[某种东西]，而[我的理论
视角]提示我们，它实际上是一种[另一种东西]。

这三个盲区并非细节遗漏，而是关系到我们如何理解[核心问题]的根本性
分歧。填补这些盲区不仅需要新的数据，更需要[理论视角]所提供的概念工具
来重新框定问题本身。
```

---

### §6 从缺口到假设（约 200–300 字）

**功能**：将综述的批判性分析导向用户选定的假设，展示综述如何"必然地"推出该假设。

**注意**：这里只讨论用户选择的那个假设（在 PHASE 3 中选定），不是提出新假设。

**写作策略**：
- 从§5的盲区出发
- 指出：如果[理论视角]是正确的，那么我们应该观察到[某模式]
- 明确这是用户选择的假设 H[X]

**段落模板**：
```
基于上述批判性综述，我们提出假设 H[X]：[假设内容]。这一假设直接回应
§5中指出的[盲区]：如果[理论机制]确实是理解[现象]的关键，那么[预测
内容]。这一预测与现有文献中的[某主张]形成[对比/补充]，因为它假设
[不同机制]而非[文献中的默认机制]。检验这一假设需要[数据/方法]，
这正是§3中指出的中文文献尚未充分开发的领域。
```

## 写作质量检查清单

完成综述后，逐一检查：

### 批判性标准
- [ ] 综述中没有任何一段是单纯的"作者X发现了Y"的罗列
- [ ] 每段都包含至少一个"但是""然而""更具批判性的是"类的转折
- [ ] 至少展示了 3 组文献之间的直接张力或矛盾
- [ ] 至少指出了 2 个文献中"被过度引用但从未检验"的概念

### 中英文平衡
- [ ] 中文文献占比 ≥ 40%（按引用数量计）
- [ ] 至少有一处明确的中英文文献对话（中文文献修正/反驳/扩展了英文文献）
- [ ] 中英文文献不是分在两个段落中，而是交织在同一论证中

### 理论视角
- [ ] 学生已选的理论视角在§4中作为核心透镜出现
- [ ] 该视角的核心概念在综述中被使用至少 3 次
- [ ] 综述展示了该视角如何"重新框定"现有文献的理解

### 论证连贯性
- [ ] §1 的 puzzle 在§5 中得到回应（不是换了另一个问题）
- [ ] §2–§4 的每个论点都在§5 的"综合"中被引用
- [ ] §6 的假设可以从§5 的盲区直接推导（不是突然跳出来的）
- [ ] 全文没有"话题突转"（abrupt topic pivot）

### 字数与格式
- [ ] 总字数 2000–3000 字（中文字符）
- [ ] 没有 bullet list（除本指南外）
- [ ] 所有引用使用作者-年份格式
- [ ] 段落之间有逻辑过渡句
