# 中文文献搜索策略指南

> 本指南供 `perspective-to-literature` skill 调用。目标：在中英文双语搜索中确保中文文献占比不低于 40%，避免中文文献沦为点缀。

---

## 核心原则

1. **双语并行**：每个搜索角度必须同时运行英文和中文查询，不能先搜完英文再"补"中文。
2. **渠道互补**：不同中文数据库各有盲区，必须交叉验证。
3. **质量优先**：中文文献必须包含核心期刊（CSSCI、北大核心）和权威著作，不接受低质量来源充数。
4. **中英文对话**：特别关注中文文献是否引用了英文核心文献、是否提出情境化修正、还是完全独立发展。

---

## 搜索渠道与查询语法

### 渠道一：百度学术（开放，覆盖面广）

**适用场景**：快速定位中文论文标题、作者、发表年份，获取引用关系。

**查询模板**：
```
site:xueshu.baidu.com "[中文关键词]" [次级关键词]
```

**示例**：
- `site:xueshu.baidu.com "幽灵外卖" 食品安全 监管`
- `site:xueshu.baidu.com "平台经济" 劳动 算法`
- `site:xueshu.baidu.com "演员网络理论" 翻译 监管`

**注意事项**：
- 百度学术返回的是元数据（标题、作者、摘要片段），通常不含全文
- 用于快速扫描：某主题下有哪些中文作者在做什么
- 记录每轮的命中数量和保留数量

---

### 渠道二：中国知网 CNKI（Site 搜索，部分开放）

**适用场景**：获取 CNKI 上已索引的论文元数据和部分开放获取全文。

**查询模板**：
```
site:cnki.net "[中文关键词]" [限定词]
site:cdmd.cnki.com.cn "[关键词]"  [学位论文限定]
```

**示例**：
- `site:cnki.net "网络餐饮" 食品安全 治理`
- `site:cnki.net "平台监管" 地方政府 执法`
- `site:cdmd.cnki.com.cn "外卖平台" 监管 硕士`

**高级技巧**：
- 加入 `PDF` 或 `filetype:pdf` 可提高获取全文的概率（部分期刊提供开放获取 PDF）
- 加入期刊名可限定来源质量：`site:cnki.net "幽灵外卖" "公共管理学报"`
- 学位论文库（cdmd.cnki.com.cn）通常对实证研究覆盖更深入

---

### 渠道三：万方数据

**适用场景**：补充 CNKI 盲区，尤其医学、工程、政策类文献。

**查询模板**：
```
site:wanfangdata.com.cn "[中文关键词]"
```

**示例**：
- `site:wanfangdata.com.cn "外卖骑手" 劳动关系`
- `site:wanfangdata.com.cn "食品安全监管" 数字化`

---

**查询模板**：

```
site:cqvip.com "[中文关键词]"
```

### 渠道五：核心期刊官网（直接搜索）

以下期刊在各自官网提供部分开放获取内容，可直接搜索：

| 期刊 | 官网搜索关键词 | 适用领域 |
|------|---------------|----------|
| 《社会学研究》 | `site:shxyj.org "[关键词]"` 或 `"社会学研究" [关键词]` | 社会学理论、经验研究 |
| 《社会》 | `"社会" 期刊 [关键词]` | 城市、劳动、家庭 |
| 《公共管理学报》 | `"公共管理学报" [关键词]` | 监管、治理、政策执行 |
| 《中国行政管理》 | `"中国行政管理" [关键词]` | 政府治理、公共政策 |
| 《管理世界》 | `"管理世界" [关键词]` | 平台经济、组织 |
| 《开放时代》 | `"开放时代" [关键词]` | 城乡、基层治理 |
| 《青年研究》 | `"青年研究" [关键词]` | 青年、教育、就业 |

**注意**：官网搜索通过 WebSearch 执行，不是直接访问数据库。用于确认某期刊是否发表过某主题的文章及作者信息。

### 渠道六：中国社会科学网 / 政策文件

**适用场景**：获取政策背景、权威解读、官方统计。

**查询模板**：
```
site:cssn.cn "[关键词]"  
site:gov.cn "[关键词]" 政策 通知
```

**示例**：
- `site:gov.cn "网络餐饮" 食品安全 管理办法`
- `site:gov.cn "外卖平台" 监管 责任`

---

### 渠道七：Google Scholar 中文搜索

**适用场景**：Google Scholar 同样索引大量中文学术论文，可作为补充。

**查询模板**：
```
site:scholar.google.com "[中文关键词]" [英文关键词]
```

**示例**：
- `site:scholar.google.com "幽灵外卖" ghost kitchen`
- `site:scholar.google.com "平台监管" platform regulation China`

## 中英文对照搜索矩阵

对于每个核心搜索角度，必须同时构造中英文查询：

| 搜索角度 | 英文查询 | 中文查询 |
|----------|----------|----------|
| 核心主题 | `"ghost kitchen" "food safety" regulation` | `"幽灵外卖" 食品安全 监管` |
| 机制 | `"platform algorithm" incentive compliance` | `"平台算法" 激励 合规` |
| 治理/政策 | `"platform governance" China enforcement` | `"平台治理" 中国 执法` |
| 理论视角 | `"actor-network theory" regulation translation` | `"行动者网络理论" 监管 翻译` |
| 方法/数据 | `"ghost kitchen" panel data OR survey` | `"幽灵外卖" 问卷调查 数据` |
| 最新前沿 | `"ghost kitchen" OR "dark kitchen" after:2020` | `"幽灵外卖" 2020 2021 2022 2023 2024` |
| 争议/批评 | `"platform regulation" debate critique` | `"平台监管" 争议 问题` |

**规则**：每个角度中英文查询各运行至少一轮。如果中文查询返回结果明显少于英文，应尝试同义词扩展（如"幽灵外卖"→"无堂食外卖"→"网络餐饮"→"纯外卖店"）。

## 搜索质量检查清单

完成所有搜索后，检查：

- [ ] 中文搜索查询数量 ≥ 英文搜索查询数量的 60%
- [ ] 至少使用了 3 个不同的中文渠道（百度学术、CNKI、万方/维普/期刊官网中的至少3个）
- [ ] 中文核心期刊文献（CSSCI/北大核心）占比 ≥ 50% 的中文保留文献
- [ ] 至少找到 1 篇中文文献明确引用/对话了英文核心文献（或反之）
- [ ] 至少找到 1 篇中文文献提出了与英文文献不同的结论或修正
- [ ] 搜索日志中每轮中文查询都有命中数和保留数记录

## 常见中文文献盲区（需主动填补）

| 盲区 | 表现 | 填补策略 |
|------|------|----------|
| 政策文件过多，实证过少 | 搜索返回大量政府文件、通知、解读 | 主动加入"实证""案例""调查"等限定词；转向学位论文库 |
| 同主题中文文献互相引用，与英文脱节 | 中文文献圈地自萌 | 检查中文文献的英文参考文献列表；反向搜索被引英文文献 |
| 英文文献主导理论，中文只做应用 | 中文文献沦为"在中国验证西方理论" | 寻找提出修正、反驳或本土概念的中文文献 |
| 缺少方法论文献 | 中文文献重描述轻方法 | 搜索特定方法关键词 + "方法""设计""效度" |
| 灰色文献缺失 | 媒体报道、白皮书、行业报告未纳入 | 加入 `"白皮书" "报告" "行业"` 等限定词 |
